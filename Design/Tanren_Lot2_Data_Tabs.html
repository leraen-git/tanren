Day-breakdown grid — clé de lecture

La grille 7 jours raconte le rythme hebdo en un coup d'œil. Vert = séance faite (volume logged), rouge avec fond teinté = aujourd'hui, gris transparent = à venir ou repos. Le nom de la séance (Push, Pull, Leg) tient dans la cellule en typo minuscule bold. Pas d'overflow : si le nom est trop long, l'app tronque à 4 caractères + ellipse.

Coaching tips — signature Tanren

Les coaching tips du brand book sont ici rendus avec le langage de la forge : "Forge +2,5 kg" quand le user progresse 3 séances d'affilée, "Trempe ton volume" quand il stagne, "Retour à l'enclume" s'il régresse. Max 1 instance par écran comme définit dans CLAUDE.md — ici c'est le call-to-action principal donc ça porte son poids.
